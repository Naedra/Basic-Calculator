# Basic-Calculator

This project is a basic calculator application developed in Java. 
It performs basic arithmetic operations such as:

- Substraction
- Adition
- Division
- Square root
- Multiplication
- Vector Dot Product(using the angle and using only two components)
- Interscion
- Union
- Power

This project also calcuate basic unit measures conversion such as:
- Temperature (Farenheit, Celcius, Kelvin)
- Length(Feet, kilometers)
